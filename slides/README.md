# 📊 Slides

This folder contains all presentation materials used for REDTEECON.

| File | Description |
|---|---|
| `Redteecon-Slides.pptx` | Final version delivered at the workshop |
| `Wireless_Hacking_slides.pptx` | Earlier draft |
| `Wireless_Hacking_FYP_slides.pptx` | FYP presentation slides (project overview, methodology, feedback) |

## Topics Covered

1. **Disclaimer & Ethical Considerations** — legal boundaries, responsible disclosure
2. **Basics of Wireless Hacking** — how wireless networks work, basic terminology, security protocols overview, tools introduction
3. **WPA** — architecture, TKIP encryption, PSK vulnerabilities, brute force attack methodology
4. **WPA2** — improvements over WPA, four-way handshake, PMKID caching, PMKID attack
5. **WPA2 Enterprise** — 802.1x, RADIUS server, CHAP/EAP protocols, Evil Twin attack
6. **Conclusion** — good wireless security habits, defensive recommendations
